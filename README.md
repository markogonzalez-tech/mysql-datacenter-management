## mysql-datacenter-management
Relational database design for a Data Center Management System using MySQL.

## MySQL Data Center Management Database

This project contains the design of a relational database for managing a data center infrastructure.

# Project Overview

The database models different components of a data center environment, including:

- Zones
- Cooling systems
- Racks
- Servers
- Users and roles
- Measurement records

# Concepts Applied

- Relational database design
- Primary and foreign keys
- ON DELETE and ON UPDATE rules
- ENUM data types
- Unique constraints
- Index creation for performance optimization
- One-to-many relationships

# Technologies Used

- MySQL
- InnoDB Engine
- MySQL Workbench

# File Included

- `datacenter_schema.sql` → Full database schema definition
